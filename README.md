# Dotfiles: Void Linux + Niri + Gruvbox

A minimal, scrollable tiling setup powered by Niri on Void Linux, styled with the cozy Gruvbox theme.
Dotfiles are managed by [chezmoi](https://www.chezmoi.io/).

![Screenshot of my Desktop](assets/screenshot.png)

## Stack & Aesthetics

- **OS:** [Void Linux](https://voidlinux.org/)
- **Window Manager:** [Niri](https://github.com/YaLTeR/niri) (Scrollable Tiling for Wayland)
- **Bar:** [Waybar](https://github.com/Alexays/Waybar)
- **Color Scheme:** [Gruvbox](https://github.com/morhetz/gruvbox)
- **Cursor:** [Capitaine Cursors](https://github.com/sainnhe/capitaine-cursors) (Gruvbox variant)
- **GTK Theme:** [Gruvbox-GTK-Theme](https://github.com/Fausto-Korpsvart/Gruvbox-GTK-Theme)
- **Icons:** [Papirus](https://github.com/PapirusDevelopmentTeam/papirus-icon-theme)
- **Terminal:** [Alacritty](https://alacritty.org/)
- **Shell:** [fish](https://fishshell.com/)
- **Editor:** [Helix](https://helix-editor.com/)
- **File Managers:** [Broot](https://dystroy.org/broot/) (TUI) & [Thunar](https://docs.xfce.org/xfce/thunar/start) (GUI)
